# AdventureX 正式 LaTeX 简历设计

## 目标

生成一份两页 A4 中文正式简历，用于 AdventureX 2026 申请。它需要同时建立医学科研可信度和产品交付能力，但不能像作品集、宣传册或个人品牌海报。

## 结构

### 第一页

1. 姓名、定位与完整联系方式
2. 个人概述
3. 教育背景
4. 科研经历
5. 核心项目：scrna-omics

### 第二页

1. 产品与团队经历：TimeBox、镜子、GoGoWork
2. 内容与社群运营（单条补充经历）
3. 专利、荣誉与项目立项
4. 技术能力与语言

## 视觉

- A4、两页、单栏、白底。
- 正文黑灰，唯一强调色为深蓝。
- 章节使用标准中文名称和细分隔线，不使用序号、英文眉题、彩色卡片或数据看板。
- 页眉保留小尺寸正式照片；个人网站二维码放在联系方式区域并缩小。
- 日期统一右对齐；角色、机构和项目名称建立固定层级。
- 正文不小于 9.5pt，页边距不小于 16mm。

## 内容取舍

- 删除 Token、模型数、调用量和 `13 / 6 / 12 / 2` 指标条。
- scrna-omics 在正文中说明阶段确认、方法选择、复核和两个真实项目，不突出内部组件数量。
- 删除社畜小猫和小产品索引。
- 自媒体只保留选题、内容生产和社群运营能力，不把曝光数字作为主视觉。
- GoGoWork 明确为创业团队项目；论文明确为在投；创新创业项目明确为校级。
- 增加已授权国家发明专利《基于人工智能的生物医学大数据分类方法及系统》，表述为“主要发明人之一”，并列出授权公告号 CN120524301B。
- 不使用“公共影响”“核心构建者”“跨界证明”等宣传式标题。

## 技术方案

- 源文件：`resume/latex/chen_sihan_adventurex_resume.tex`
- 编译器：本机 `tectonic`
- 输出：`output/pdf/陈思翰_AdventureX_正式简历.pdf`
- 预览：`tmp/pdfs/formal-resume/`
- 自动检查：两页 A4、中文可提取、必需事实存在、禁用宣传与 Token 文案。

## 验收

- 编译无错误、无 overfull box。
- PDF 恰好两页 A4，文本可复制。
- 两页在 PNG 预览中无裁切、重叠、孤行和异常留白。
- 联系方式、教育、科研、项目、荣誉、技能均能按阅读顺序提取。
- 旧 ReportLab 简历及生成器保留，新的正式版使用独立文件名。
